Easy-Root

A Swift framework for simplifying radicals.

Usage

CocoaPods

Add EasyRoot to your Podfile:

pod 'EasyRoot'

And run pod install

Features

Currently supports simplifying integer-only radicals. For example:

  • Root 4
    • Result: 2
  • 3 root -27
    • Result: -3 cube roots of 3

And so forth.
